Директория = Папка


pwd - Показывает текущее местонахождение в папке 
ls - Показывает папки и файлы которые находятся в текущей директории(папке)
mkdir [название папки] - Создает папку с указанным названием 
touch [название файла] - Создает файл с указанным названием
cd [название папки] - Перемещает в папку с указанным названием 
cd .. - Возвращает в предыдущую папку
cd - Возвращает в Домашнюю Директорию из любого места  

cp [файл] [куда копировать] - Копирует файл в указанное место
mv [файл] [куда переместить] - Перемещает или переименовывает файл
rm [файл] - Удаляет файл
rmdir [папка] - Удаляет пустую папку
rm -rf - Удаляет папку вместе с содержимым 
cat [файл] - Показывает содержимое файла
echo [текст] - Выводит текст в терминал
clear - Очищает экран терминала
whoami - Показывает имя текущего пользователя
man [команда] - Показывает справку по команде
exit - Закрывает текущую сессию терминала
